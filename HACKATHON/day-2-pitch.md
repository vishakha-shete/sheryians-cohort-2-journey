Floating label inputs
Error handling with animations
Loading states

Created reusable components:

AuthLayout → Split screen layout (branding + form)
InputField → Floating label + validation UI
Button → Animated, variant-based button
SocialAuth → Social login section

👉 This ensures scalability and maintainability.
